Demostración de Web APIs

Explora ejemplos prácticos de las principales APIs del navegador.

DOM

Manipula elementos HTML, nodos y eventos para crear interfaces dinámicas e interactivas.

Canvas

Dibuja gráficos, permite elegir color y grosor de línea, y crea ilustraciones interactivas en 2D.

Cámara

Accede a la cámara del dispositivo, elige entre varias y muestra el video en tiempo real.

Almacenamiento

Guarda y recupera datos usando LocalStorage, SessionStorage y Cookies de forma sencilla.

Red

Realiza peticiones HTTP con Fetch y prueba comunicación en tiempo real con WebSocket.

Geolocalización

Obtén la ubicación actual del usuario usando la API de geolocalización del navegador.

Archivos

Selecciona archivos locales y muestra información básica como nombre, tipo y tamaño.

Notificaciones

Solicita permiso y muestra notificaciones nativas del sistema desde el navegador.

Seguridad

Consulta permisos y explora la API de gestión de credenciales del navegador.
